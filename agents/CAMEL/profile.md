# CAMEL

## Overview

CAMEL (Communicative Agents for "Mind" Exploration) is an open-source
multi-agent framework that facilitates role-playing interactions
between AI agents to accomplish tasks collaboratively.

## Key Information

- **Developer:** CAMEL-AI community
- **Website:** [https://github.com/camel-ai/camel](https://github.com/camel-ai/camel)
- **Pricing:** Open-source (Apache-2.0)

## Key Features

- **Large-scale agent system:** Supports simulations of up to millions of
  agents for scaling-law research.
- **Dynamic communication:** Enables real-time collaboration among role-playing agents.
- **Stateful memory:** Agents retain context across multi-step interactions.
- **Data generation and tool integration:** Automates dataset creation
  and integrates external tools.

## Supported Models

- OpenAI GPT-4 family
- Anthropic Claude models
- Meta Llama and Mistral variants
- Other platforms via CAMEL's unified ModelType enumerations
